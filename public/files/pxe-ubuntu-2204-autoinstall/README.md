# Ubuntu 22.04 PXE autoinstall templates

These files are sanitized templates from the PXE installation note.

Replace every placeholder before use:

- `<PXE_SERVER_IP>`
- `<PXE_INTERFACE>`
- `<TARGET_MAC>`
- `<TARGET_HOSTNAME>`
- `<INSTALL_USERNAME>`
- `<SHA512_CRYPT_PASSWORD_HASH>`
- `<WIFI_INTERFACE>`
- `<WIFI_STATIC_IP_CIDR>`
- `<LAN_GATEWAY>`
- `<LAN_DNS>`
- `<WIFI_SSID_5G>`
- `<WIFI_SSID_FALLBACK>`
- `<WIFI_PASSWORD>`

Do not publish real sudo passwords, login passwords, Wi-Fi passwords, or full device identifiers.
